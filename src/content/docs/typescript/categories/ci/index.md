Utility functions for working with ci operations.

## Functions

| Function | Description |
|----------|-------------|
| [`buildStatusTable`](./buildstatustable/) | Builds a Markdown table body from a map of job names to CI/CD statuses. |
| [`DEFAULT_PERCENTAGE_TIERS`](./default_percentage_tiers/) | Default tiers, geared towards coverage/quality-gate style percentages. |
| [`percentageToTier`](./percentagetotier/) | Maps a numeric percentage to a tier (icon, color, label) using configurable thresholds. |
| [`statusToBadge`](./statustobadge/) | Maps a CI/CD job status to an inline code badge string. |
| [`statusToIcon`](./statustoicon/) | Maps a CI/CD job status to an emoji icon. |
