Utility functions for working with url operations.

## Functions

| Function | Description |
|----------|-------------|
| [`addLeadingSlash`](./addleadingslash/) | Adds a leading slash `/` if not yet present. |
| [`extractPureURI`](./extractpureuri/) | Extracts only the URI from a path, dropping any query and fragment. |
| [`removeEndingSlash`](./removeendingslash/) | Removes an ending slash `/` if present. |
