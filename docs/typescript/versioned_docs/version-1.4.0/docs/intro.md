# Version 1.x (Legacy)

This documentation covers version 1.x of the TypeScript Helpers library.

:::warning
Version 1.x is now in maintenance mode. We recommend upgrading to [version 2.x](../version-2.x).
:::

## Key Features

- Legacy array, date, and string utilities
- CommonJS and ES module support
- Node.js 14+ support

## Upgrade Guide

See the [Upgrade Guide](../migration/from-v1) to migrate from v1 to v2.
