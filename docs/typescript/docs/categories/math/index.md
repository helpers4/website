# Math Helpers

Utility functions for working with math operations.

## Functions

| Function | Description |
|----------|-------------|
| [`uuid7`](uuid7) | Generates a UUID v7 string (RFC 9562). UUID v7 embeds a Unix timestamp in milliseconds, making it chronologically sortable while retaining randomness. |
