# clamp

Clamps a number between min and max values

## Import

```ts
import { clamp } from '@helpers4/number';
```

## Signature

```ts
function clamp(value: number, min: number, max: number): number
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| `value` | The value to clamp |
| `min` | Minimum value |
| `max` | Maximum value |

## Returns

Clamped value
