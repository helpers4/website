# Function Helpers

Utility functions for working with functions.

## Functions

| Function | Description |
|----------|-------------|
| [`debounce`](debounce) | Creates a debounced function that delays invoking func until after delay milliseconds have elapsed since the last time the debounced function was invoked |
| [`isDefinedAndNotNull`](isDefinedAndNotNull) | Check if a given value of unknown data type is defined and not null |
| [`memoize`](memoize) | Returns a memoized version of the function that caches results |
| [`returnOrThrowError`](returnOrThrowError) | Return a value or throw an error is null or undefined. |
| [`throttle`](throttle) | Creates a throttled function that only invokes func at most once per every wait milliseconds |
