# isDate

Checks if a value is a Date

> Available since v1.9.0

## Import

```ts
import { isDate } from '@helpers4/type';
```

## Signature

```ts
isDate(value: unknown): value
```

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `value` | `unknown` | The value to check |

## Returns

`value` — True if value is a Date
