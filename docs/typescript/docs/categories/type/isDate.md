# isDate

Checks if a value is a Date

## Import

```ts

```

## Signature

```ts
function isDate(value: unknown): value is Date
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| `value` | The value to check |

## Returns

True if value is a Date
