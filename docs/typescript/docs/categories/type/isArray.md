# isArray

Checks if a value is an array

## Import

```ts
import { isArray } from '@helpers4/type';
```

## Signature

```ts
function isArray(value: unknown): value is unknown[]
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| `value` | The value to check |

## Returns

True if value is an array
