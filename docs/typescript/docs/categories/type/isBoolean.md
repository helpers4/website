# isBoolean

Checks if a value is a boolean

## Import

```ts
import { isBoolean } from '@helpers4/type';
```

## Signature

```ts
function isBoolean(value: unknown): value is boolean
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| `value` | The value to check |

## Returns

True if value is a boolean
