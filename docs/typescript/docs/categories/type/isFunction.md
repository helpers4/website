# isFunction

Checks if a value is a function

## Import

```ts
import { isFunction } from '@helpers4/type';
```

## Signature

```ts
function isFunction(value: unknown): value is Function
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| `value` | The value to check |

## Returns

True if value is a function
