# isString

Checks if a value is a string

## Import

```ts
import { isString } from '@helpers4/type';
```

## Signature

```ts
function isString(value: unknown): value is string
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| `value` | The value to check |

## Returns

True if value is a string
