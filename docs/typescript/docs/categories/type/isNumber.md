# isNumber

Checks if a value is a number

## Import

```ts
import { isNumber } from '@helpers4/type';
```

## Signature

```ts
function isNumber(value: unknown): value is number
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| `value` | The value to check |

## Returns

True if value is a number
