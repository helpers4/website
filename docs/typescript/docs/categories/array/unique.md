# unique

Removes duplicate values from an array

## Import

```ts
import { unique } from '@helpers4/array';
```

## Signature

```ts
function unique<T>(array: T[]): T[]
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| `array` | The array to remove duplicates from |

## Returns

New array with unique values only
