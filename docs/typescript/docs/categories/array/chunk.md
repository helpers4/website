# chunk

Chunks an array into smaller arrays of specified size

## Import

```ts
import { chunk } from '@helpers4/array';
```

## Signature

```ts
function chunk<T>(array: T[], size: number): T[][]
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| `array` | The array to chunk |
| `size` | The size of each chunk |

## Returns

Array of chunks
