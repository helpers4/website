# isSameDay

Checks if two dates are the same day

## Import

```ts
import { isSameDay } from '@helpers4/date';
```

## Signature

```ts
function isSameDay(date1: Date, date2: Date): boolean
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| `date1` | First date |
| `date2` | Second date |

## Returns

True if same day
