# daysDifference

Gets the difference in days between two dates

## Import

```ts
import { daysDifference } from '@helpers4/date';
```

## Signature

```ts
function daysDifference(date1: Date, date2: Date): number
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| `date1` | First date |
| `date2` | Second date |

## Returns

Number of days difference
