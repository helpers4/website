# dateToISOString

Formats a date to ISO string or returns null

## Import

```ts
import { dateToISOString } from '@helpers4/date';
```

## Signature

```ts
function dateToISOString(input: string | number | Date | null | undefined): string | null
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| `input` | Date input |

## Returns

ISO string or null
