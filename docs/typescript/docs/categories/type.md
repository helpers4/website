# Type

Type guards and runtime type checking utilities.

## Import

```typescript
import { /* type utilities */ } from '@helpers4/typescript';
```

## Overview

Type utilities provide runtime type checks and guards that work seamlessly with TypeScript's type system, enabling safe narrowing of unknown values.
