# Version Helpers

Utility functions for working with versions.

## Functions

| Function | Description |
|----------|-------------|
| [`compare`](compare) | Compares two semantic version strings according to SemVer 2.0.0 specification |
| [`increment`](increment) | Increments a semantic version |
| [`parse`](parse) | Parses a semantic version string into its components according to SemVer 2.0.0 specification |
| [`satisfiesRange`](satisfiesRange) | Checks if a version satisfies a range (simple implementation) |
| [`stripV`](stripV) | Strip the leading "v" from a version string if it exists. |
