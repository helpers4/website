# deepClone

Creates a deep copy of an object or array

## Import

```ts
import { deepClone } from '@helpers4/object';
```

## Signature

```ts
function deepClone<T>(obj: T): T
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| `obj` | The object to clone |

## Returns

Deep cloned object
