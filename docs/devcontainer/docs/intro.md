# DevContainer Features

This is the documentation for **helpers4 DevContainer Features** - a collection of development container features for modern web development.

## Features

Each feature provides a complete development environment setup for specific development needs:

- [vite-plus](./features/vite-plus.md) - Advanced Vite toolchain configuration
- [typescript-dev](./features/typescript-dev.md) - TypeScript development environment
- [git-absorb](./features/git-absorb.md) - Git absorb integration
- [shell-history-per-project](./features/shell-history-per-project.md) - Per-project shell history
- [local-mounts](./features/local-mounts.md) - Local mount configuration
- [angular-dev](./features/angular-dev.md) - Angular development environment
- [package-auto-install](./features/package-auto-install.md) - Automatic package installation

## Installation

Features are available from: `ghcr.io/helpers4/devcontainer/<feature-name>`

Learn more in the [GitHub repository](https://github.com/helpers4/devcontainer).
