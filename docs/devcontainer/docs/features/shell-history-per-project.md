# shell-history-per-project

Persist shell history per-project in your dev container.

## Overview

This feature preserves your shell command history separately for each project, so you always have relevant history available when reopening a dev container.

## Installation

```json
{
  "name": "Shell History",
  "features": {
    "ghcr.io/helpers4/devcontainer/shell-history-per-project:latest": {}
  }
}
```
