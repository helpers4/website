# angular-dev

Angular development environment with pre-configured tooling and extensions.

## Features

- **Angular CLI** - Latest Angular CLI pre-installed
- **TypeScript** - TypeScript compiler configured for Angular
- **VS Code Extensions** - Angular language service and debugging tools

## Installation

```json
{
  "name": "Angular Dev",
  "features": {
    "ghcr.io/helpers4/devcontainer/angular-dev:latest": {}
  }
}
```
