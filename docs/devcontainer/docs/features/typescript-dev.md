# typescript-dev

Complete TypeScript development environment with pre-configured tooling and VS Code extensions.

## Features

- **TypeScript 5.x** - Latest TypeScript compiler
- **Node.js 20+** - Modern Node.js runtime
- **VS Code Extensions** - TypeScript IntelliSense, error checking, and debugging
